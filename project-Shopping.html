Project Shopping Application

Project presentation:

The goal of the project is to write a shopping cart application which will enable the customer to make the required purchase.
The application will also support the business organization to maintain its records

Image:

Project operations:

  • Make purchases as a customer (manage their basket, make purchases, view history).
  • Manage the site as an employee (history, invoices, statistics) shoe website.
  • offers different brands of shoes, different types (sneakers, heels, slippers, etc.)

Used Technology:

Regarding the implementation part of the functionalities we had to use the "Java" programming language and for the graphical aspect of the project we could used "GUI" both on Apache NetBeans IDE 12.0 or NetBeans IDE 8.2.

For everything that was user data management, ie client data, we had to set up a database using the software / language of our choice.
I opted for "MySQL Workbench 8.0 CE" software for the simple reason that it is the database management software that I know and master the best.
